# CUSTOM IGG IMAGE LIB

## Implement these functions to Image:
+ ScaleUp
+ ScaleDown
+ Histogram
+ Cols
+ Rows
+ FillFromPgm
+ WriteToPgm
+ At(row,col)

## Build
Create the folder to build
	```
	mkdir build
	cd build
	```
	Build CMake
	```
	cmake ..
	make
	```
## Run main program
	```
	./bin/main
	```
## Run Test program
	```
	./bin/test_img
	```
