## Implement this in Image:
+ Create a PPM Strategy to write/read/modify PPM images.
+ Build a CMAKELIST.txt to PngIoStrategy

## Build
Create the folder to build
```
mkdir build
cd build
```
Build CMake
```
cmake ..
make
```

## Run main program
```
./bin/main
```
## Run Test program
```
./bin/color_image_test
```
